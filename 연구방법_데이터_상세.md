# 연구방법·데이터 상세

[연구계획서](연구계획_교통상충_TTC_PET_임계값_전이성.md)의 실행 문서. 계획서가 "무엇을 왜", 이 문서가 "어떻게"를 정한다. **5절 판정 기준은 분석 착수 전에 확정해야 한다.**

## 1. 데이터 제약과 대응
데이터 확보 완료 (85개 파일 14GB, MD5 검증 통과).

| 제약 | 대응 (계획서 §8) |
|---|---|
| 신호현시 정보 없음 | TTC는 추종, PET는 교차·회전으로 역할 분담 |
| 제공 속도가 사전 필터링용 | 좌표에서 재계산 |
| `Road_Section` 내부 74% 결측 | 외곽 섹션 OD 매칭으로 이동류 분류 |
| 교차로 L·P 좌표오차 1m 초과 | 두 곳 제외 민감도 |
| 호버링 분절 (파일 15분 중 유효 세션당 중앙값 11분) | 상충률 분모는 실측 관측시간, PET는 세그먼트 내부만 |
| PM 세션 과반이 정지 상태 | 정지차량 필터 |

표본 규모(교차로 G 실측): 20m 이내 근접쌍 세션당 280~1,460쌍, 교차로-일당 3,000~8,000쌍.
**TAAS**: 2020~2024 5년 합산. 반경은 관측영역(약 198×197m)의 절반인 **50m를 기본**으로 하고 30·100m를 민감도로 둔다. 자료를 보기 전에 확정한다.

## 2. TTC·PET 산출 — Espadaler-Clapés et al. (2025) 기준
우리 데이터를 만든 팀이 같은 파이프라인으로 계산한 논문이라 기준으로 삼는다.
```
TTC = dist(sub, surr) / (v_sub − v_surr),  단 v_sub > v_surr
PET = t²c − t¹c   (t^x_c = 차량 x가 공통 지점을 통과한 시각)
```
원문 조건: 차량을 점으로 봄(길이 미반영). TTC는 거리가 줄어드는 추종 전용. 교차 판정은 방위각 차이 30도 이상.
우리가 바꾸는 것: 차량 치수 반영을 민감도로 추가, 속도 재계산, PET를 세그먼트 내부 한정, 이동류는 OD 매칭.
노출량 보정: 이동류쌍 통행량 곱 × 유효 관측시간으로 나눈 상충률.

## 3. 임계값
**고정값** — TTC 1.5초(주분석). PET는 통용 대표값이 확인되지 않아 단일 값을 지정하지 않고 스윕 곡선 전체를 결과로 제시(계획서 §8). 스윕은 0.1초 간격이며, 계획서 §2에서 확인한 **문헌 사용 범위 0.5~6.0초를 반드시 포함**한다.
**자료 기반값** — 각 교차로 상충 분포의 하위 p 분위수, 주분석 p=5%, 스윕 1~25%.
**p=5% 정당화** — 직접 선례 없음. "선행연구를 따랐다"고 쓰지 말 것. ① 분위수를 임계값으로 쓰는 방식은 선례 있음(Al-Haideri 2025, Jiao 2024) ② 지점별 임계값은 표준 관행(Arun 2022) ③ 5%는 "사전 지정한 최소 사건 수 규칙을 만족하는 가장 엄격한 수준"으로 정의.
**핵심 방어 논거** — 우리 임계값은 EVT의 임계값과 역할이 다르다. 저쪽은 꼬리 근사용 보조모수, 우리 것은 상충 여부를 가르는 판정 컷오프. 요건은 둘뿐(셀당 사건 수 확보, 지점 간 동일 상대 엄격도).
**풀링** — 교차로별 임계값의 중앙값(주분석), 사건 풀링 분위수(민감도).
**순환성 주의** — 각 교차로를 자기 5%로 자르면 정의상 모두 5% 상충을 갖는다. 상충 비율 비교는 동어반복이므로 1차 지표는 임계값 이동량·판정 일치도·순위 상관.

## 4. 검증 설계
**LOIO 20-fold (주분석, 계획서 RQ3)** — ① 19곳 10/4~06으로 통합 임계값 ② 남긴 교차로 소량 자료로 보정 임계값 ③ 둘 다 남긴 교차로 10/7에 적용 ④ 무보정 대 보정 비교. 20회 반복.
**보정 사다리 (RQ4)** — 1세션(중앙값 11분) → 3세션(약 30분) → 1일(약 110분) → 3일(약 330분). 어느 세션을 뽑느냐로 결과가 달라지므로 무작위 추출을 반복하고 분포를 보고한다.
**잡음 바닥** — 같은 교차로 자료를 무작위로 반 갈라 한쪽 임계값을 다른 쪽에 적용, 반복. 진짜 차이가 0이므로 여기서 나오는 불일치가 **표본 잡음의 기준선**이다. 계획서 H2·H3이 "통계 잡음 이상", "판정이 유지되지 않는다"를 말하므로 **이 기준선이 두 가설의 판정 근거가 된다.** 임의 상수를 쓰지 않기 위해 반드시 먼저 계산한다.
**쌍별 전이 (보조)** — 한 교차로 값을 나머지 19곳에 적용, 20×19=380회. 가설도 합격 기준도 걸지 않고 히트맵으로만 제시, RQ5 재료.
**시간 축** — "시간적 전이성" 대신 "익일 재현성의 사례 평가"로 명명. forward-chaining 3회.
**통계** — 분석 단위는 세션, 군집 강건 추론 또는 교차로 랜덤효과. 순위는 부트스트랩 CI 병기하고 CI가 겹치는 차이는 해석하지 않는다. 셀당 상충이 한 자릿수면 세션을 통합한다(경계값은 사건 수 표를 본 뒤 확정).

## 5. 가설 판정 기준 ★ 사전 지정
계획서 §5의 가설 문구를 그대로 조작화한 것이다. 결과를 보기 전에 확정하고 `config.yaml`에 동결한다.
**원칙**: 임의 상수를 최소화하고, 가능하면 잡음 바닥에서 얻은 자료 기준선에 건다. 불가피한 관행값은 관행임을 밝히고 연속값을 함께 보고해 결론이 그 값에 걸리지 않게 한다.

| 가설 (계획서 §5) | 판정 기준 | 기준의 출처 |
|---|---|---|
| **H1** 임계값을 바꾸면 순위가 실질적으로 뒤바뀐다 | 문헌 사용 범위(0.5~6.0초) 안의 두 값 사이에서 교차로 순위 Kendall τ < 0.8이 발생. 보조로 상충률 변화 배수의 교차로 간 범위 > 2배 | 범위는 계획서 §2에서 확인한 문헌 산포. τ 0.8은 관행값이며 스윕 곡선 전체를 병기 |
| **H2** 교차로마다 임계값이 **통계 잡음 이상으로** 다르다 | 교차로 간 임계값 랜덤효과 SD의 95% CI 하한 > 잡음 바닥에서 얻은 교차로 **내** SD | 가설 문구를 그대로 조작화. 임의 상수 없음 |
| **H3** 전이하면 판정이 유지되지 않는다 | LOIO의 판정 불일치(이동량·Jaccard·τ)가 잡음 바닥 분포의 상위 5%를 벗어남. 즉 같은 교차로 내 반분할로는 설명되지 않는 수준 | 잡음 바닥 기준. 임의 상수 없음 |
| **H4** 소량 현지 보정이 무보정보다 오차를 줄인다 | 20 fold에서 \|보정오차\|가 현지 보정 < 무보정 (교차로 단위 짝지은 단측 Wilcoxon, α=0.05). 중앙값 차이와 부트스트랩 CI 병기 | α=0.05는 관행값, 효과크기 병기로 보완 |
| **H5** 안정성이 지표·조건에 따라 다르다 | 보정오차 혼합모형에서 지표×시간대, 지표×상충유형, 지표×교차로형태 교호작용 ≠ 0 (군집 강건 추론, 효과크기 보고) | 조건 네 축은 계획서 RQ5·H5와 동일 |
| **H6** 상충 기반 등급이 사고 등급과 우연 이상으로 일치 | 상충 순위와 TAAS 사고 순위의 Spearman·Kendall 상관 + 아래 3갈래 해석 | 계획서 §4 RQ6에서 정한 사전 지정 해석 |

**H6 3갈래 해석** — ① 일치(상관 뚜렷, CI가 0 미포함) → 타당성 근거 확보 ② 불일치(상관 낮음이 분명) → 원인 고찰 ③ 판별불가(건수 부족으로 CI 넓음) → "송도 규모 신도시에서는 5년 자료로도 교차로 간 판별이 어렵다"로 보고. 계획서 §2-1의 실증이 된다.
**H6 등급 전환** — 5년 합산 사고 중앙값이 3건 미만이면 순위 대신 상·중·하 등급 비교(계획서 §4 RQ6). 각 순위를 3등분(7/6/7)하되 동점은 같은 등급. 지표는 상등급 적중률이며, 우연 수준은 초기하분포 기대값으로 병기한다.
**주의** — 잡음 바닥에 거는 기준(H2·H3)은 잡음 바닥을 **먼저 계산해야** 확정된다. 착수 순서를 지킬 것.

## 6. 하지 말 것
- **TAAS 사고 설명력이 최대가 되는 값을 임계값으로 채택** — RQ6이 순환논증이 된다(Park 2025 방식). 사후 대조로만.
- **EVT 진단도표(MRL·stability plot)를 근거로 사용** — GPD를 적합하지 않으므로 기준 없는 그림이 된다.
- **원문 미확보 문헌 인용** — 남의 각주에서 본 것을 읽은 것처럼 쓰지 않는다.

## 7. 착수 전 확인
1. 세션 구조를 20개 교차로 전수 실측 → 계획서 §8 수치 확정
2. 교차로-세션별·교차로-일별 실현 사건 수 표 → p=5%가 세션 단위에서 버티는지 판정, 부족하면 추정 단위를 교차로-일로 상향. 논문 본문 표로도 싣는다
3. TTC 1.5초가 우리 자료에서 몇 번째 백분위인지 → 선택 기준이 아니라 보고 항목
4. p=1·5·10%에서 분위수 추정치 흔들림 부트스트랩
5. **잡음 바닥 계산** → H2·H3 판정 기준이 여기서 확정된다
6. TAAS 3~4개 교차로 정찰 조회로 건수 규모 확인

## 8. 인용 한계선 (원문 확인 결과)
- **Gettman & Head 2003**: TTC 1.5초는 `user determined parameter, e.g., 1.5 s`인 예시값이며 van der Horst(1990) 인용. **PET 임계값 없음**
- **Arun 2021 (AAP 153, 106016)**: 549편 검토. `thresholds are chosen arbitrarily ... without justification`. 후미추돌 TTC가 0.5~6.0초 산포, PET도 동일 문제
- **Espadaler-Clapés 2025 (TRIP 32, 101490)**: 임계값에 `no solid conclusion has been reached`라 밝히며 값 채택. 맨체스터 2곳
- **Borsos 2020**: 단일 지점 한계 자인
- **Park 2025**: SSRN 프리프린트(게재본 AAP DOI 10.1016/j.aap.2025.108167 미확보). 개별 모형 기각하고 3곳 통합 채택, 공개 CitySim, 미국 올랜도
- **윤일수 외 2011**: 한국ITS학회논문지 10(6), 1-14. **임계값 명시 없음** — "국내에서도 이 도구가 쓰였다"까지만
- **Hydén 1987**: 이미지 스캔이라 대조 불가. Tarko(2018)는 기여자 중 하나로, Arun(2021)은 안전 연속체 모형 출처로 서술 — 그 범위에서만 인용
